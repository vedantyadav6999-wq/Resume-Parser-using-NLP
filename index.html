// 05 — ML Model

Career Prediction Engine

Trained on real-world job profile datasets, our ML model delivers high-accuracy career path recommendations.

🧠 Model Specs

AlgorithmRandom Forest Classifier
Dataset10,000+ labeled job profiles (Kaggle)
FeaturesSkills, experience, certifications, education
OutputTop-3 career role predictions + confidence
Accuracy~87% on test dataset
VectorizerTF-IDF (Term Frequency-Inverse Doc Freq)
Inference< 2 seconds per resume
APIServed via REST API (Flask)

🎯 How It Works

  • 1
    Text Extraction
    PyPDF2 / python-docx extracts raw text from resume
  • 2
    NLP Preprocessing
    spaCy tokenizes, lemmatizes, removes stopwords
  • 3
    Feature Engineering
    TF-IDF vectors encode skill terms and context
  • 4
    Model Prediction
    Random Forest outputs top 3 career role probabilities
  • 5
    Result & Report
    Score, gaps, and career paths presented in dashboard
